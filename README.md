Fundraising Website

A simple and responsive fundraising platform developed to support social causes and charitable campaigns.

🌟 Features

- User Registration and Login
- Browse Fundraising Campaigns
- About Us Page
- Contact Page
- Responsive User Interface
- Clean and Simple Design

🛠️ Technologies Used

- HTML5
- CSS3

📁 Project Structure

fundraising/
│── index.html          # Home page
│── about.html          # About page
│── campaigns.html      # Campaigns page
│── contact.html        # Contact page
│── login.html          # Login page
│── signup.html         # User registration page
│── styles.css          # Styling file
│── README.md           # Project documentation

🚀 Getting Started

1. Clone the repository:

  git clone https://github.com/varshitha211/fundraising.git

2. Navigate to the project folder:

cd fundraising

3. Open "index.html" in your browser.
